安卓布局中的dp

  • Android

dp是安卓布局中最常见的单位，它表示dip，一种基于屏幕密度的抽象单位。在每英寸160点（160dpi）的显示器上，1dp = 1px。
dp与px之间的换算要跟dpi关联，不同dpi的设备下的1dp对应的像素数也不相等。

dp与px的换算公式如下：
dp = ( DPI / 160 ) * px
英寸与像素的换算关系如下：
英寸 = px / DPI
